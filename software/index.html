Freie Open Source Software für den Desktop

Einfache Begriffserklärungen

Die folgenden Aussagen sind technisch gesehen vielleicht nicht ganz korrekt, sollen aber einen schematischen Überblick geben.
  • Wenn der Computer started, wird das Debian Programm ausgeführt. Debian ist ein Betriebssystem, also ein sehr complexes Programm. (Eigentlich sind es mehrere.)
  • In diesem Debian (Betriebssystem) wird nach dem Einloggen eine graphische Oberfläche gestartet, die an sich noch keine Formen und Farben hat. Die Oberfläche heisst X.
  • Das Betriebssystem startet auch ein weiteres Programm Gnome, das benutzt X um Fenster darzustellen. Gnome wird als Fenstermanager oder Desktopmanager bezeichnet.
  • Innerhalb von Gnome kann man nun weitere Programme starten, die entweder eine graphische Benutzeroberfläche haben, oder auch nicht. Man spricht hier von Applikationen. Gnome liefert einige Applikationen gleich mit. Andere Applikationen werden von anderen Projekten beigesteuert. Die folgende Tabelle stellt einige Applikationen vor.
